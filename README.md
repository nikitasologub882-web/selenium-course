# Selenium WebDriver Automation Course 

## О проекте
Курс по автоматизации тестирования с использованием Selenium WebDriver, Python и Pytest. Включает уроки от основ до продвинутых тем.

## Технологии
- Python 3.x
- Selenium WebDriver 4.x
- Pytest (для тестирования)
- WebDriver Manager (для управления драйверами)
- Pickle (для работы с cookies)
- Git/GitHub - контроль версий и хостинг кода
